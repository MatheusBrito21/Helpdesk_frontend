Helpdesk Sistema gerenciador de chamados

Olá! Seja bem vindo ao sistema helpdesk desenvolvido por mim no curso Desenvolvedor FullStack.
Esse é um sistema para gerenciamento de chamados simulando um simples Help Desk desenvolvido em Angular e Spring Boot.
O sistema está rodando na nuvem sendo o front na Vercel e o back na Heroku.
O Projeto tem como objetivo principal abordar os conceitos de aplicações SPAs, Autenticação e Autorização com Spring Security e muito mais. O Código fonte do projeto pode ser acessado através do link do Github no menu lateral.

Fique a vontade para testar o sistema e deixar sua sugestão.
